# Django Flatpages Engine

Учебный проект на Django, в котором реализован движок для управления статическими страницами (Flatpages).

## Что было сделано:
* **Безопасность:** Все секретные ключи вынесены в переменные окружения (`.env`).
* **Локализация:** Полностью настроена русскоязычная панель администратора.
* **Структура:** Проект переведен на использование современного модуля `pathlib` для работы с путями.
* **Контент:** Подключен модуль `django.contrib.flatpages` для создания страниц "О нас", "Контакты" и др. через админку.

## Как запустить:
1. Клонировать репозиторий.
2. Установить зависимости: `pip install -r requirements.txt`.
3. Создать файл `.env` и добавить туда свой `SECRET_KEY`.
4. Запустить миграции: `python manage.py migrate`.
5. Запустить сервер: `python manage.py runserver`.
